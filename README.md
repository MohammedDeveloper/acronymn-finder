- Find <[A-Z]{2,}>. (i.e., letters matching A to Z[capitalized] having minimum 2 letters)
- Use “ “ while reading words from the document
- Regular Expression: <[A-Z]{2,}>
- Get the acronyms matched based on step 4 and display
